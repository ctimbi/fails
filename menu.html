Accessibility Fails

This page contains a set of segments html code of common mistakes in web pages. It is based on the work of @dugboticus and web-page http://aduggin.github.io/accessibility-fails/. Other fails are added: experience and Failures for WCAG 2.0

Accessibility Fails are grouped in accordance to the level of compliance and success criteria according to Web Content Accessibility Guidelines (WCAG) 2.0
